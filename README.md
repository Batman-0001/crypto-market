# 🚀 Interactive Crypto Calendar

A modern, feature-rich interactive calendar application for cryptocurrency market data visualization. Built with React and Material-UI, this application provides real-time market analysis, advanced pattern recognition, comparative analysis tools, and comprehensive export capabilities - all wrapped in a beautiful, glass-morphism design.

## 🌟 What Makes This Special

- **🎨 Modern Glass-morphism Design**: Beautiful, modern UI with smooth animations and professional styling
- **📊 Advanced Pattern Analysis**: AI-powered pattern recognition for market trends and trading signals
- **⚖️ Comparison Tools**: Compare different time periods and cryptocurrencies side-by-side
- **📤 Export Everything**: Export charts, data, and analysis to PDF, CSV, and images
- **🔄 Real-time Data**: Live market data integration with automatic updates
- **📱 Fully Responsive**: Works perfectly on desktop, tablet, and mobile devices

### 🔧 Troubleshooting Common Issues

#### **Data Not Loading**

1. Check your internet connection
2. Try refreshing the page
3. Check if Binance API is accessible in your region
4. Clear browser cache and reload

#### **Export Not Working**

1. Ensure you have data loaded for the selected period
2. Check browser popup blockers
3. Try downloading with a different browser
4. Contact support if issues persist

#### **Pattern Analysis Empty**

1. Ensure sufficient historical data is available
2. Try different cryptocurrencies or time periods
3. Pattern recognition requires at least 30 days of data
4. Some patterns are rare and may not appear frequently

### �🚀 Getting Started

### Installation & Setup

#### **Prerequisites**

- Node.js (version 16 or higher) - [Download here](https://nodejs.org/)
- npm or yarn package manager (comes with Node.js)
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for real-time data

#### **Quick Setup (5 minutes)**

1. **Clone or Download the Project**:

   ```bash
   # If you have git
   git clone <repository-url>
   cd interactive_calendar

   # Or download and extract the ZIP file
   ```

2. **Install Dependencies**:

   ```bash
   npm install
   # This will download all required packages (~2-3 minutes)
   ```

3. **Start the Application**:

   ```bash
   npm run dev
   ```

4. **Open in Browser**:
   - Navigate to `http://localhost:5173`
   - The app will load with Bitcoin data automatically
   - No additional configuration needed to get started!

#### **Production Build**

```bash
# Build for production
npm run build

# Preview production build
npm run preview

# The build will be in the 'dist' folder
```

### 🔧 Configuration (Optional)

The app works out of the box, but you can customize it:

#### **Environment Variables**

Create a `.env` file in the root directory for custom settings:

```bash
# API Configuration
VITE_CRYPTO_API_BASE_URL=https://api.binance.com/api/v3
VITE_DEFAULT_CRYPTO_SYMBOL=BTCUSDT
VITE_REFRESH_INTERVAL_TICKER=10000

# Feature Flags
VITE_ENABLE_DEBUG_LOGGING=false
VITE_ENABLE_WEBSOCKET_UPDATES=true

# For advanced users (optional)
VITE_BINANCE_API_KEY=your_api_key_here
VITE_BINANCE_SECRET_KEY=your_secret_key_here
```

## ⌨️ Keyboard Navigation

- **Arrow Keys**: Navigate between dates
- **Ctrl + Arrow Keys**: Navigate by weeks/months
- **Enter/Space**: Select focused date
- **Escape**: Cancel selection
- **Home**: Go to first day of month
- **End**: Go to last day of month
- **Page Up/Down**: Navigate months
- **Shift + Page Up/Down**: Navigate years
